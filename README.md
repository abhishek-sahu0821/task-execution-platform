# Task Execution Platform

A distributed task execution platform built with FastAPI, PostgreSQL, Kafka, and Kubernetes.

## Tech Stack
- FastAPI
- PostgreSQL
- Kafka
- Redis
- gRPC
- Docker
- Kubernetes

## Setup
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```
